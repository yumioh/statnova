## 1. 프로젝트 개요
### 주제
- **우리는 왜 통계데이터과학과를 선택했고, 어떻게 생존할 것인가?**
- AI 열풍이 학과 선택과 취업 시장 역량(Tableau, PowerBI 등)에 미친 영향 분석

### 분석 배경
- AI 산업의 급성장에 따른 통계학의 위상 변화
- 단순 통계를 넘어 데이터 사이언스로의 패러다임 전환 확인

## 3. 분석 시나리오 및 데이터 수집
  3.1 다음 뉴스와 네이버 뉴스 API 활용의 한계 
  초기 단계에 다음 뉴스와 네이버 뉴스 API 검토하였으나, 다음과 제약 사항으로 인해 제외
   - 네이버 API 검색의 한계 : 파타미터 제약으로 인해 키워드당 최대 1000건까지만 접근 가능
   - 네이버 API 기간 검색 불가능 : 특정 날짜 범위를 지정하는 파라미터가 없음
   - 다음 뉴스 : 각 언론사별의 사이트별로 연결되어 상이한 HTML 구조를 일일이 대응하기 어려워 효율성이 낮음
<br>

    3.2 Selenium 활용
    API의 한계를 극복하고 분석의 신뢰도를 높이기 위해 Selenium 기반의 웹 크롤러를 사용 
    - API 제한없이 일자별로 뉴스 헤드라인 및 본문 수집 가능
