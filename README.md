## 3. 분석 시나리오

### 3.1 데이터 수집 도구 검토 및 한계점
초기 데이터 수집 단계에서 **다음(Daum) 뉴스**와 **네이버(Naver) 뉴스 API**를 검토하였으나, 다음과 같은 기술적 제약으로 인해 최종 수집 도구에서 제외하였습니다.

* **네이버 뉴스 API 검색의 한계**
    * `start` 파라미터 제약으로 인해 검색어당 **최대 1,000건**까지만 접근 가능하여 대규모 데이터 확보가 어려움.
    * 특정 날짜 범위를 지정하는 파라미터가 부재하여, 시계열 트렌드 분석을 위한 수집이 불가능함.
* **다음(Daum) 뉴스 수집의 한계**
    * 검색 결과가 각 언론사별 개별 사이트(Outlink)로 연결되어 있어, **상이한 HTML 구조**에 언론사별 개별 파싱이 필수적인 기술적 난점


### 3.2 Selenium 기반 데이터 수집 전략
API의 제약을 극복하고 규모를 확보하기 위해 **Selenium 기반의 웹 크롤러와 BeautifulSoup을 활용.
* **일자별 수집:** 네이버 뉴스 검색의 기간 설정 기능을 Selenium으로 제어하여 API 제한 없이 일자별 뉴스 데이터를 수집.
* **뉴스 구조 일관성 확보:** '네이버 뉴스(In-link)' 형식의 기사만 필터링, 텍스트 데이터의 구조적 일관성을 유지함.

### 3.3 채용 데이터 수집 및 분석
* **대상:** 워크넷 API (https://www.work24.go.kr/cm/e/a/0110/selectOpenApiSvcInfo.do?apiSvcId=000000000000000000000000000060&upprApiSvcId=000000000000000000000000000059&fullApiSvcId=000000000000000000000000000000%5E000000000000000000000000000001)
* **내용:** 통계 전공자 우대 조건, 직무 공고 추이, 요구 기술 스택(Python, SQL, Tableau 등) 변화 분석.

### 3.4 교육/학과 데이터 분석
* **대상:** 공공데이터포털, 교육부(KESS) 학과 통계 (https://kess.kedi.re.kr/contents/dataset?itemCode=04&menuId=m_02_04_03_02&tabId=m2)
* **내용:** 학과별 입학 인원 및 경쟁률 변화를 통해 AI 산업 성장이 교육 수요에 미친 상관관계
