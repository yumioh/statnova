## 1. 프로젝트 개요
### 주제
- **우리는 왜 통계데이터과학과를 선택했고, 어떻게 생존할 것인가?**
- AI 열풍이 학과 선택과 취업 시장 역량(Tableau, PowerBI 등)에 미친 영향 분석

### 분석 배경
- AI 산업의 급성장에 따른 통계학의 위상 변화
- 단순 통계를 넘어 데이터 사이언스로의 패러다임 전환 확인
